# Codex Notes

This directory stores working context for future Codex sessions.

It is intentionally kept outside `content/` and `static/`, so Hugo does not
publish it as a site page. The files are still part of the GitHub repository if
they are committed.

Suggested use:

- `project-context.md`: stable facts about the site and repo.
- `history/`: dated summaries of design decisions, fixes, and open items.

Do not store secrets, tokens, private email contents, or unpublished sensitive
data here. This repository may be public.
